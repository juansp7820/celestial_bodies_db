# Celestial Bodies DB
From the 'Relational Databases Certification' at freecodecamp.org, this database is a certification project in which one builds a SQL database featuring many methods meant to build a database related to astronomy.  

## Featuring
Basic methods of data-architecture in SQL, such as CREATE TABLE, FOREIGN KEYS as constraints, and so on. The contents of this SQL database include basic info about celestial bodies such as planets. 

## References
The sources of these contents were retrieved from wikipedia.org and nasa.gov as for June 9th, 2026.
